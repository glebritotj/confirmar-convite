# Sistema de Confirmação de Convites

Página pública para oradores confirmarem presença em discursos públicos.

## Como Funciona

1. Sistema gera link único para cada convite
2. Orador acessa o link
3. Escolhe uma das datas oferecidas
4. Confirma presença
5. Sistema sincroniza automaticamente com Google Sheets

## Tecnologias

- HTML5 + CSS3
- JavaScript
- Google Sheets API
- Responsivo (funciona em celular)

## Deploy

Hospedado no GitHub Pages para acesso mundial.
